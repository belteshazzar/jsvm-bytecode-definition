# jsvm-bytecode-definition

This package defines the bytecode contract shared by all runtimes.

Contents:
- `docs/FORMAT.md`
- `docs/OPCODES.md`
- `docs/VALUES.md`
- `src/version.js` (`BYTECODE_VERSION`)
- `src/io.js` (binary encode/decode)

This package has no dependency on the compiler or any runtime.
